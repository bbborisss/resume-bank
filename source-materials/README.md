# source-materials/

Drop your raw material here before running `/create-resume-bank`:

- Your current resume, in any format (PDF, Word, plain text)
- Old resumes or past versions
- LinkedIn profile export or pasted profile text
- Brag documents, performance reviews, self-assessments
- Old cover letters
- Portfolio or project links

`/create-resume-bank` reads everything in this folder along with anything you attach
directly to the conversation. Nothing here needs to be tidy — dumping raw, overlapping,
even redundant material is fine; the setup interview sorts it out with you.

This folder's contents are yours, not the template's — see the repo's `.gitignore` for
guidance on keeping it out of version control if your copy of this repo is public.
