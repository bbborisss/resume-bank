# Tailoring preferences

Rules `/tailor-resume` applies to every run, accumulated from your feedback.
Each rule is a short imperative with a one-line why. Rules here **override the
skill's defaults**. `/tailor-resume` appends to this file when you give feedback
that should stick; edit or delete rules freely.
